Descrição detalhada

Minimal example: mount SD, append a line to a file, unmount.

Este exemplo demonstra o fluxo essencial de uso do FatFs:

  • Obter a instância sd_card_t configurada (ver hw_config.*)
  • Montar o volume com f_mount
  • Abrir/criar arquivo e escrever usando f_open/f_printf
  • Fechar com f_close e desmontar com f_unmount
