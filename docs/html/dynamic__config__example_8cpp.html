Descrição detalhada

Example showing how to build SPI and SD card configuration dynamically.

Instead of compiling a fixed hw_config.c, este exemplo aloca e preenche os objetos spi_t e sd_card_t em tempo de execução, registra-os no sistema e realiza uma operação simples de E/S de arquivo via FatFs.

Passos:

  • Inicializa stdio/tempo
  • Cria e configura spi_t (pinos, baud)
  • Cria e configura sd_card_t (SS, Card Detect, nome lógico)
  • Monta o volume, escreve uma linha em filename.txt e desmonta

Útil para cenários nos quais os pinos variam entre placas ou revisões.
