Exemplo: CLI FatFS via SPI (RP2040)

Exemplo interativo com linha de comando (CLI) para exercitar operações do FatFs sobre cartão SD via SPI no Raspberry Pi Pico (RP2040).

  • Comandos: format, mount, unmount, ls, cat, mkdir, cd, big_file_test, start_logger, stop_logger, help, etc.
  • Código-fonte principal: example/FatFS_SPI_example.cpp

Compilar

cd example
mkdir -p build && cd build
cmake ..
make -j

O UF2 será gerado em build/.

Gravar no Pico

  1. Entre em BOOTSEL (manter botão pressionado ao conectar USB)
  2. Arraste o .uf2 gerado para a unidade montada

Executar

  • Abra um terminal serial (ex.: tio -m ODELBS /dev/ttyACM0)
  • Pressione Enter para ver o prompt >
  • Digite help para listar os comandos e exemplos de uso

Configuração de Hardware

  • O mapeamento de pinos/placa é definido em example/hw_config.c e variantes (ex.: thing_plus.hw_config.c).
  • Verifique os pinos de SPI, CS (SS) e, opcionalmente, Card Detect conforme sua ligação.

Notas

  • Para segurança ao trocar cartões, use Card Detect ou desmonte antes de remover.
  • Reduza o baud_rate do SPI se houver instabilidade.
  • Gerado por 1.9.8
