Descrição detalhada

FatFs helpers: error-to-string and recursive directory deletion.

Implementa utilitários usados pelos exemplos/testes:

  • FRESULT_str(FRESULT) converte códigos FRESULT em mensagens legíveis
  • delete_node() remove recursivamente diretórios e arquivos de um caminho

Mantém a licença original; nenhuma lógica funcional foi alterada.

Documentação das funções

◆ delete_node()

FRESULT delete_node ( TCHAR *  path,
UINT  sz_buff,
FILINFO *  fno 
)

Recursively delete a directory tree.

Deletes the directory at path and all of its contents (files and subdirectories). The caller must provide a buffer of size sz_buff for path manipulation and a FILINFO structure.

Parâmetros
pathPath buffer containing the directory to delete (mutable)
sz_buffSize of the path buffer in elements (TCHAR items)
fnoPointer to a FILINFO structure used during traversal
Retorna
FRESULT FR_OK on success or a FatFs error code

Referências AM_DIR, delete_node(), f_closedir(), f_opendir(), f_readdir(), f_unlink(), FILINFO::fattrib, FILINFO::fname e FR_OK.

Referenciado por delete_node().
