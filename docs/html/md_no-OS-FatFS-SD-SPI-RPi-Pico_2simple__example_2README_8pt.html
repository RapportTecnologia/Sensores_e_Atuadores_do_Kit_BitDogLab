Exemplo Simples (RP2040 + FatFs SPI)

Exemplo mínimo que monta o cartão SD, adiciona uma linha a filename.txt e desmonta.

Fonte: simple_example/simple_example.cpp

Compilar

cd simple_example
mkdir -p build && cd build
cmake ..
make -j

Gravar e Executar

  • Grave o UF2 gerado (modo BOOTSEL, arrastar e soltar)
  • Abra um terminal serial para acompanhar os logs

Notas

  • Garanta que hw_config.c corresponde à sua ligação de pinos (SPI, CS/SS e, opcionalmente, Card Detect)
  • Reduza o baud_rate do SPI em caso de instabilidade
  • Gerado por 1.9.8
