Descrição detalhada

Interface C++ simples para o periférico I2C do RP2040/RP2350.

Esta classe funciona como um pequeno wrapper ao redor das funções do SDK do Pico, oferecendo uma API semelhante ao Wire/Arduino para facilitar exemplos e drivers de sensores.
